# BUG-002: Checkout Form Accepts Special Characters in Name Fields

**Severity:** Medium | **Priority:** Medium

### Description
The "First Name" and "Last Name" fields in the checkout process allow symbols like `@#$%^&*`, which should be restricted.

### Steps to Reproduce
1. Add any item to the cart and proceed to Checkout.
2. In the "First Name" field, enter `Shaza@123`.
3. Complete the form and click "Continue".

### Expected Result
Validation error should appear: "Name fields can only contain alphabetic characters."

### Actual Result
The system accepts the special characters and proceeds to the Checkout Overview page.
