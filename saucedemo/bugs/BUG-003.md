# BUG-003: Authentication Bypass via Browser "Back" Button

**Severity:** High | **Priority:** High

### Description
After logging out, clicking the browser's "Back" button allows the user to view cached authenticated pages.

### Steps to Reproduce
1. Login to SauceDemo.
2. Click "Logout" from the sidebar menu.
3. Once redirected to the login page, click the browser's "Back" button.

### Expected Result
The user should be redirected to the login page or prompted to log in again (session should be destroyed).

### Actual Result
The user can view the previous session's product inventory page.
